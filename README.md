# [Qase TMS](https://qase.io) Robot Framework Listener

# Installation

```
pip install qase-robotframework
```

# Contribution

Install project locally:

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -e .[testing]
```

Install dev requirements:

```bash
pip install pre-commit
pre-commit install
```

Test project:

```bash
tox
```
